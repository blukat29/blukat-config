blukat-config
=============

blukat-config is a collection of config files.

Installation
------------

```sh
git clone https://github.com/blukat29/blukat-config.git
cd blukat-config
sh basic.sh
```

This requires following programs:
- ctags (exuberant-ctags)

If you are too lazy to install them, just run
```sh
sudo sh full.sh
```
